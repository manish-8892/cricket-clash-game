# 🏏 Hand Cricket Simulator

A simple menu-driven cricket game built using Python. This project simulates a basic hand cricket match between the user and the computer using random number generation and core programming concepts.

---

## 📌 Features

* Toss system (Heads/Tails)
* Choice to Bat or Bowl
* Ball-by-ball gameplay
* Score tracking and wicket system (3 wickets)
* Target-based match system
* Input validation (only numbers 1–6 allowed)
* Menu-driven interface (Play / Exit)

---

## 🛠️ Technologies Used

* Python (Core Concepts)
* Random Module


   ```
   cd hand-cricket-simulator
   ```
4. Run the program:

   ```
   python main.py
   ```

---

## 🎮 How to Play

1. Start the game from the menu.
2. Choose Heads or Tails for toss.
3. Select Batting or Bowling if you win the toss.
4. Enter numbers between 1 to 6 for each ball.
5. If your number matches the computer → OUT.
6. Otherwise, runs are added.
7. After first innings, chase the target.
8. Final result will be displayed.
